# Land of Legends AI Lore

This folder holds AI-assisted or AI-generated lore drafts for the `Land of Legends` module.

## Rules

- Mirror the canonical lore tree structure
- Keep AI drafts separate from approved canonical lore
- Promote reviewed content into the parent `lore/` tree when accepted
