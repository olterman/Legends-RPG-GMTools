# Land of Legends Lore

This is the **central lore repository** for the `Land of Legends` module.

## Purpose

Use this folder for reusable, canonical module lore.

- Structural manifests stay with the world nodes
- Rich prose lives here
- The folder tree mirrors the module structure

## Canonical Lore

Canonical approved lore belongs directly under this `lore/` root:

```text
lore/
  overview.md
  history.md
  culture.md
  religion.md
  politics.md
  relationships.md
  secrets.md
  peoples/
  regions/
  creatures/
  items/
  system/
```

## AI Lore

AI-assisted or AI-generated drafts belong under:

```text
lore/ai_lore/
```

That tree should mirror the canonical lore structure, but remain separate until its contents are reviewed and promoted.

## Campaign Material

Campaign-specific files do **not** belong here.

Keep these in the campaign layer instead:

- `gm_notes.md`
- `adventure_hooks.md`
- `session_notes.md`
- `reveals.md`
- `changes.md`
