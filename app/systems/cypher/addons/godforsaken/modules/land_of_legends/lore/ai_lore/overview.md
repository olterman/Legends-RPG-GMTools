# AI Lore

The AI lore branch is the module's staging ground for machine-assisted working material.

Nothing here is canonical by default. This branch exists so prompts, drift checks, triage notes, and AI-first drafts can live close to the setting without being mistaken for approved lore.

Use this branch when you need to:

- prepare art-direction prompts
- capture drift tests for fragile setting concepts
- triage races, cultures, and places before promoting them into canonical lore
- hold AI-assisted drafts that still need review

Promote reviewed material into the parent `lore/` tree when it becomes canon.
