# Drift Test: Daelgast Erosion

Use this when Daelgast-related content starts sounding like standard demonic corruption.

## Red Flags

- Daelgast is personified as a villain with plans
- corruption is framed only as evil magic
- undead are treated as simple servants
- Hollow Accord material reads like infernal cult material

## Correction Move

Return to erosion, incompatibility, unresolved passage, failed release, contamination, and mistaken revelation.
