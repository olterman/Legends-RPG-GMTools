# GMForge Branding Assets

This folder stores product-level branding assets for `GMForge`.

Use this location for:
- app logos
- wordmarks
- launcher icons
- brand-ready export variants

Recommended filenames:
- `gmf_logo.svg`
- `gmf_logo.png`
- `gmforge_wordmark.svg`

These assets belong to the core product identity, not to any specific system, addon, or campaign.
