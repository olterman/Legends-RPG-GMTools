# Land of Legends Module

`land_of_legends` is the first setting module scaffold under the `godforsaken` Cypher expansion.

This folder is for shipped, system-owned module content, not user campaign data.

Suggested usage:
- authored setting metadata and module manifests live here
- shipped lore, NPCs, items, maps, and indexes can live here later
- user-created campaign data belongs in `content/`
