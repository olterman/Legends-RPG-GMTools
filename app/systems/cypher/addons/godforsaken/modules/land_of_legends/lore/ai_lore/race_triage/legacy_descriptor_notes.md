# Legacy Descriptor Notes

The legacy descriptor file mixes several layers together:

- canonical people identity
- Cypher-facing mechanics
- optional GM cultural traits
- taxonomy choices that do not perfectly match the rebuild

This is useful source material, but it should be treated as a bundle that needs controlled splitting.

Important current tensions:

- `Velim` appears in the legacy descriptor file under Small Folk, but in the rebuild taxonomy it lives under `The Others`.
- `Lhainim` descriptor variants in the legacy file include names like `Boggart` and `Satyr` that do not currently exist in the rebuild subgroup tree.
- the descriptor file is closer to a system design packet than a pure lore file, so promotion should continue under `lore/system/` rather than being scattered through race overviews.
