# Godforsaken Addon

`godforsaken` is a Cypher expansion addon.

It is not a setting itself. It is the expansion layer that can later own settings and modules such as `land_of_legends`.

Artifacts in this folder:
- raw source in `source/`
- Docling markdown output in `source_markdown/`
- Docling html output in `source_html/`
- addon metadata in `addon.json`
- readable rulebook metadata in `rulebook.json`
